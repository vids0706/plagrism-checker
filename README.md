# Plagiarism Checker
A full-stack web app capable of identifying plagiarism in uploaded files.
## What the app can do- User authentication (login and registration)- Student page for submitting documents- Instructor page for viewing works submitted- Detection of plagiarism and display of similarity results
## What technologies have been used- Node.js- Express.js- HTML- CSS- JavaScript
## Project Structure- Controllers to implement the business logic- Routes to define API endpoints- Static files to serve the frontend UI
## What can be done in the future- Use sophisticated NLP technology for similarity detection- Enhance plagiarism accuracy- Make database optimization
